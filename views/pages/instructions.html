Agendor Front-end Engineer Quiz

What is this?

It's an interview test to evaluate your programming logic while you create an algorithm to solve a problem.

How does it work?

After you press "Start" button, you'll have 60 minutes to solve a problem using Javascript.

There will be a code editor on the next page that you must use to write code.

There will be buttons to run your code and submit your solution. You can submit your solution more than once.

Are there any rules?

  • The time counting that start on the next page can't be paused
  • You must write your code using our code editor

How do we evaluate?

We don't look just to your final solution, we want to see who did you get there. That's possible with a session recording. When you get to the next page, your screen in this page will be recorded. That's why you must use our code editor to write code.

We will look for:

  • Your algorithm abilities
  • Your data structure abilities
  • How elegant is your solution
  • The quality of your final solution code
